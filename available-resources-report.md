## 8. 제한사항 및 주의사항

1. **gemini-analyze-image**: 로컬 파일 경로 필요. URL 이미지 직접 분석 불가 → WebFetch로 먼저 다운로드 필요
2. **이미지 편집 세션 (start/continue/end)**: 기존 사진을 업로드해서 편집하는 것이 아니라, 새로 생성한 이미지를 반복 수정하는 구조. "기존 사진 보정"보다 "스타일 학습 후 새 이미지 생성"에 적합
3. **nano-banana-pro**: Gemini API 래퍼. 결과 품질은 100% Gemini 모델에 의존
4. **Memory MCP**: 세션 간 데이터 유지되지만, 복잡한 쿼리는 불가. 단순 키워드 검색만 지원
5. **GEMINI_API_KEY**: 무료 티어 사용 시 분당/일일 호출 제한 있을 수 있음
