# 섹션 구성 가이드

## 핵심 원칙

선택된 레퍼런스의 섹션 구성을 먼저 확인하라.
레퍼런스의 `report.md`에 섹션 목록과 순서가 있다.
아래 가이드는 레퍼런스에 없는 섹션을 추가하거나 순서를 조정할 때 참고한다.

---

## 1. 섹션 순서 (방문자 감정 여정)

```
1. 네비게이션 바 (고정)
2. 히어로/메인 비주얼        ← 호기심
3. 숫자로 증명 (선택)        ← 놀라움
4. 메뉴/상품                ← 식욕/관심
5. 브랜드 소개              ← 신뢰
6. 가맹 안내 (핵심)          ← 확신
7. 인테리어/매장             ← 확신 강화
8. 본사 지원 시스템           ← 안심
9. 성공 사례/후기            ← 결심
10. 개설 절차 (선택)         ← 구체화
11. 문의/상담 (CTA)          ← 행동
12. 푸터
```

순서 논리: 호기심 → 놀라움 → 식욕 → 신뢰 → 확신 → 행동

---

## 2. 섹션 우선순위

### 필수 (실측 70%+)

| 섹션 | 사용률 | 핵심 |
|------|--------|------|
| 가맹 안내 | 87% | 창업 비용, 조건, CTA |
| 인테리어/매장 | 83% | 매장 사진 갤러리 |
| 메뉴/상품 | 78% | 대표 메뉴 3~6개 카드 |

### 강력 권장 (50~70%)

| 섹션 | 사용률 |
|------|--------|
| 브랜드 소개 | 61% |
| 문의/상담 | 61% |
| 히어로 | 52% |
| 본사 지원 | 52% |
| 성공 사례 | 52% |
| 푸터 | 52% |

### 차별화 (50% 미만)

| 섹션 | 사용률 | 차별화 효과 |
|------|--------|-----------|
| 수익/매출 공개 | 45% | 절반 미만 = 있으면 차별화 |
| 뉴스/미디어 | 39% | 언론 노출, 수상 이력 |
| 성공 사례 상세 | 35% | 가맹점주 인터뷰 |
| 개설 절차 | 15% | 스텝 타임라인 |

## 3. 업종별 추천 조합

### 고기/한식 (10~12개)

네비 → 히어로 → 숫자 → 메뉴 → 브랜드 → 가맹 → 인테리어 → 본사지원 → 성공사례 → 문의 → 푸터

레퍼런스: 003(10개), 004(10개), 020(10개) 참고

### 치킨/분식 (8~10개)

네비 → 히어로 → 메뉴 → 가맹 → 인테리어 → 본사지원 → 성공사례 → 문의 → 푸터

레퍼런스: 010(5개), 011(6개) — 이보다 풍부하게 구성 권장

### 카페 (8~10개)

네비 → 히어로 → 메뉴 → 브랜드 → 가맹 → 인테리어 → 본사지원 → 문의 → 푸터

레퍼런스: 019(11개, 전체 최다), 017(7개) 참고

## 4. CTA 배치 전략

### CTA 개수: 3~5개 (실측 평균 3.5개)

| 위치 | CTA 유형 | 목적 |
|------|----------|------|
| 네비 우측 | 버튼 | 항상 보이는 가맹 문의 |
| 히어로 | 큰 버튼 | 첫 화면 행동 유도 |
| 가맹 안내 하단 | 버튼 | 비용 확인 후 바로 문의 |
| 문의 섹션 | 폼 + 전화번호 | 최종 전환 |
| 모바일 하단 (고정) | 전화 + 문의 | 스크롤 중 접근 가능 |

### 상담폼: 5~7개 필드 권장

```
이름, 연락처, 희망지역 (필수 3개)
+ 창업예산, 문의내용 (선택 2개)
```

레퍼런스별 필드 수: 022(5), 009(7), 020(7), 001(14), 023(15)

### 플로팅 CTA (모바일)

```html
<div class="fixed bottom-0 left-0 right-0 z-50 md:hidden bg-[배경주] border-t border-[보더색] p-3 flex gap-[20px]">
  <a href="tel:[전화번호]" class="flex-1 text-center py-3 border border-[액센트] text-[액센트] font-bold
    transition-all duration-300 ease-out">
    전화 상담
  </a>
  <a href="#contact" class="flex-1 text-center py-3 bg-[액센트] text-white font-bold
    transition-all duration-300 ease-out">
    가맹 문의
  </a>
</div>
```
