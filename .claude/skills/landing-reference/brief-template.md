# 프로젝트 기획서 (brief.md)

이 파일은 새 랜딩페이지 프로젝트를 시작할 때 작업 폴더에 생성된다.
AI가 사용자에게 질문하거나, 사용자가 직접 채워서 완성한다.
`[미입력]`인 항목은 AI가 물어본다. 이미 채워진 항목은 넘어간다.

| 슬로건 | [미입력] |

## 사용법

```
1. 이 파일을 작업 폴더에 brief.md로 복사
2. 아는 항목을 먼저 채운다 (모르는 건 [미입력]으로 둔다)
3. AI에게 "기획서 채워줘" 또는 "랜딩페이지 만들어줘"라고 말한다
4. AI가 [미입력] 항목을 하나씩 물어본다
5. 다 채워지면 AI가 레퍼런스를 선택하고 제작을 시작한다
```
