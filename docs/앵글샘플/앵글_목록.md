| **양식/파인다이닝** | 클래식 45° | 로우앵글 30° | 입체감 + 프리미엄 느낌 |

## 프리셋 사용 가이드

고객이 앵글을 선택할 때:
1. **모르겠으면 → 45° 클래식** (가장 무난, 실패 없음)
2. **상차림/세트 → 90° 탑뷰** (여러 메뉴 한눈에)
3. **층이 있는 음식 → 0° 아이레벨** (버거, 케이크, 음료)
4. **질감 강조 → 매크로** (초밥, 스테이크, 빵)
5. **프리미엄 느낌 → 30° 로우앵글** (웅장하고 고급)
